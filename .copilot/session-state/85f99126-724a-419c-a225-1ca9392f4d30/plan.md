# Agent Break — Gameplay Feature Plan

## Features (priority order)
1. Elevated enemy platforms (block bridges + enemies on top)
2. Piranha plants (pop up/down from pipes)
3. Warp pipes (press ↓ to teleport ahead)
4. Enemy spawner pipes
5. Desktop dive pipe (overlay goes transparent)
6. Parachute drop mode (float from top, dodge flying enemies)
